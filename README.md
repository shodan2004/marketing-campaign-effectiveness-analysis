# Marketing Campaign Effectiveness Analysis

An end-to-end machine learning project that analyzes marketing campaign data to predict customer response and generate actionable business insights.

The project covers data preprocessing, feature engineering, model training using a RandomForestClassifier, and an interactive Streamlit dashboard for visualization and analysis.
